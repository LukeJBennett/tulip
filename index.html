tulip

What is tulip?

Simply put, tulip is an open source software project for creating cross country roadbooks designed to optimize the creation of routes in a simple, basic set of steps using an intuitive, fast, and visually succinct UI.

What do I need to use it?

Tulip currently uses the Google Maps API which requires an active internet connection for operation since it dynamically fetches the mapping data via the connection. Even though an internet connection is required to run the app, all roadbook and gpx data files are stored locally.

What does it cost?

Tulip is, and always will be, free for personal use. Our primary purpose is to promote the growth of cross country rally.

So how do I go from making a route on tulip to navigating a roadbook?

Tulip outputs roadbooks as a PDF file in Letter, A5, and Roll sizes. You can then use your PDF app of choice to modify this output to suit your needs.

Additionaly tulip can output a GPX file of your route for pre-running. This export is formatted so that when it is imported into devices and apps like Rally Comp and Rally Blitz all WPM, WPS, DSZ, ASZ, DSS, and ASS route notifications are already baked in.
